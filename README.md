# report

Создание ежедневного отчета (report.docx), два сервера на windows, несколько серверов на разных линукс. Очередь процессора, место на диске и тд. В отчет вставляются скрины с windows, текстовый вывод из линукс. Пакеты pyautogui, paramiko, docx

Если какие-то команды не успевают отработать в винде, добавить sleep()
